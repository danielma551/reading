# 优雅阅读应用

这是一个基于Next.js开发的阅读应用，使用了Apple设计风格，提供了精美的阅读体验。

## 功能特点

- 简洁优雅的阅读界面
- 深色模式支持
- 自定义背景颜色
- 阅读目标管理（每次400句）
- 句子卡片分组（每25句为一组）
- 进度指示器
- 字体大小调整
- 文件保存与管理
- 响应式设计

## 技术栈

- Next.js
- React
- Tailwind CSS

## 开始使用

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build

# 启动生产服务器
npm run start
```

## 部署

本项目可以部署在Vercel上，只需连接到GitHub仓库即可。
