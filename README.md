# 閱讀器應用

一個優雅的中文閱讀應用，帶有進度追蹤和自定義外觀。

## 特點

- 段落閱讀模式，每次顯示一個句子
- 閱讀進度跟踪系統，分為三個階段
- 字體自定義，支持導入自定義字體
- 背景顏色自定義
- 深色模式支持
- 數據同步和備份功能
- 響應式設計，適配桌面和移動設備

## 技術

基於Next.js開發

## 功能特点

- 简洁优雅的阅读界面
- 深色模式支持
- 自定义背景颜色
- 阅读目标管理（每次400句）
- 句子卡片分组（每25句为一组）
- 进度指示器
- 字体大小调整
- 文件保存与管理
- 响应式设计

## 技术栈

- Next.js
- React
- Tailwind CSS

## 开始使用

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build

# 启动生产服务器
npm run start
```

## 部署

本项目可以部署在Vercel上，只需连接到GitHub仓库即可。
